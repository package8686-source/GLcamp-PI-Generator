# GLcamp Proforma Invoice Generator

Next.js web app for creating GLcamp / Guangzhou Lvguang Trading Co., Ltd. proforma invoices.

## Run Locally

```bash
npm.cmd install
npm.cmd run dev
```

Open:

```text
http://localhost:3000
```

## Build

```bash
npm.cmd run build
```

## Main Files

```text
app/
app/page.tsx
app/api/pi-number/route.ts
app/globals.css
next.config.ts
types/html2pdf.d.ts
```
